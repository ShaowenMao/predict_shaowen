# Supporting Information workspace

`../supporting_information.tex` is the root document for the Water Resources Research Supporting Information PDF.

## Layout

- `sections/`: supporting methods and diagnostics, numbered as Text S1, S2, and so on in the contents list.
- `figures/`: publication-ready supporting figures. LaTeX numbers them Figure S1, Figure S2, and so on.
- `tables/`: optional standalone table fragments. LaTeX numbers them Table S1, Table S2, and so on.

## AGU scope rules

- Keep supporting text, figures, tables, and captions in one PDF.
- Keep the paper's key analysis and interpretation in the main manuscript.
- Deposit data and software in appropriate repositories rather than packaging them as Supporting Information.
- Add references used by the Supporting Information to the main manuscript under `References From the Supporting Information`.
- Submit the compiled PDF because AGU does not copyedit or reformat Supporting Information.

Guidance: <https://www.agu.org/publications/authors/journals/supporting-information>
