# AGU template provenance

This project vendors the files required by AGU's current LaTeX manuscript template:

- `agujournal2025.cls`
- `tweaklist-git-moderncv-fixed.sty`
- `wiley-macros.tex`
- `agu-logo-large.pdf`
- `agu-logo-small.pdf`

Source repository: <https://github.com/AGU-Publications/agujournal2025-latex-template>

Source commit: `355052226d872cf6b9211c12b73b2ed2da133a7d`

Retrieved: 19 July 2026

AGU's submission checklist describes `agujournal2025.cls` as the current template that more closely resembles the Wiley version of record and addresses issues in `agujournal2019.cls`:

<https://www.agu.org/publications/authors/journals/submission-checklists>
